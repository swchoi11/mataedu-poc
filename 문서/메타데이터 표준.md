# 개별 문항 메타데이터 표준

이 문서는 개별 문항에서 추출하고 관리하는 메타데이터의 표준을 정의합니다. `entities.py`의 `Item` 클래스와 `models.py`의 `Metadata` 클래스를 기반으로 작성되었습니다.

## 메타데이터 필드 목록

### 1. 학년 (Grade)
- DB 컬럼: `grade`
- 설명: 문제가 해당하는 학년 정보.
- 데이터 타입: `String`
- 예시: "고1", "고2", "고3"

### 2. 과목 (Subject)
- DB 컬럼: `subject`
- 설명: 문제가 해당하는 과목 정보.
- 데이터 타입: `String`
- 예시: "수학", "국어", "영어"

### 3. 추천 단원 1 (Suggested Chapter 1)
- DB 컬럼: `main_chap1`, `mid_chap1`, `small_chap1`
- 설명: AI가 추천하는 첫 번째 연계 단원 정보.
- 데이터 타입: `String`
- 구성:
    - `main_chap1`: 대단원
    - `mid_chap1`: 중단원
    - `small_chap1`: 소단원

### 4. 추천 단원 1 이유 (Reason for Suggestion 1)
- DB 컬럼: `reason1`
- 설명: 첫 번째 단원을 추천한 이유.
- 데이터 타입: `String`

### 5. 추천 단원 2 (Suggested Chapter 2)
- DB 컬럼: `main_chap2`, `mid_chap2`, `small_chap2`
- 설명: AI가 추천하는 두 번째 연계 단원 정보 (선택 사항).
- 데이터 타입: `String`
- 구성:
    - `main_chap2`: 대단원
    - `mid_chap2`: 중단원
    - `small_chap2`: 소단원

### 6. 추천 단원 2 이유 (Reason for Suggestion 2)
- DB 컬럼: `reason2`
- 설명: 두 번째 단원을 추천한 이유 (선택 사항).
- 데이터 타입: `String`

### 7. 난이도 (Difficulty)
- DB 컬럼: `difficulty`
- 설명: 문제의 난이도.
- 데이터 타입: `Enum` (`DiffEnum`)
- 값:
    - "상": 어려움
    - "중": 보통
    - "하": 쉬움

### 8. 난이도 평가 이유 (Difficulty Reason)
- DB 컬럼: `difficulty_reason`
- 설명: 해당 난이도로 평가한 구체적인 이유.
- 데이터 타입: `String`

### 9. 문제 유형 (Item Type)
- DB 컬럼: `item_type`
- 설명: 문제의 형식.
- 데이터 타입: `Enum` (`ItemTypeEnum`)
- 값:
    - "5지선다": 5-choice multiple choice
    - "조합형": Complex choice (e.g., ㄱ, ㄴ, ㄷ)
    - "단답형": Short answer
    - "서술형": Essay / descriptive answer

### 10. 배점 (Points)
- DB 컬럼: `points`
- 설명: 문제에 부여된 점수.
- 데이터 타입: `Integer`
- 예시: 4

### 11. 출제의도 (Intent)
- DB 컬럼: `intent`
- 설명: 문제를 통해 평가하고자 하는 학생의 역량이나 지식.
- 데이터 타입: `String`

### 12. 핵심 키워드 (Keywords)
- DB 컬럼: `keywords`
- 설명: 문제와 관련된 핵심 개념 키워드. 쉼표(`,`)로 구분.
- 데이터 타입: `String`
- 예시: "삼각함수, 로그"

### 13. 문제 내용 (Content)
- DB 컬럼: `content`
- 설명: 이미지 등에서 추출된 문제의 전체 텍스트.
- 데이터 타입: `String`

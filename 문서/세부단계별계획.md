
## 설계 및 기반 구축
- 메타 데이터 표준 정의
    - 난이도 기준 확정
    - 교과서 단원 매핑 방식 확정
- db 모델링 및 구축
    - 시험지, 문항, 메타데이터 테이블 설계
- 교과 단원 테이블 및 엑셀 데이터 import

## 핵심 엔진 개발
- ocr 연동 모듈 개발
    - 파일을 받아 텍스트로 반환하는 공통 모듈(paddlepaddle/ gemini)
- 문항 분리 로직 개발
    - ocr된 전체 텍스트를 받아 개발 문항 텍스트 배열로 분리
    - gemini 활용
    - 예외 처리 로직
- gemini 메타 데이터 분석 파이프라인 개발
    - 문항 텍스트를 받아 json 스키마로 메타 데이터 반환

## 워크 플로우 구현
- 스트림릿 ui 골격 개발
    - 파일 업로드 컴포넌트, 분석 결과 표시 영역 등 
- 업로드 처리 및 분석 실행 로직 개발
    - 파일 저장
    - ocr 모듈 호출
    - 문항 분리 모듈 호출
    - 분리된 각 문항을 분석 파이프라인에 전달
    - 모든 결과를 db에 저장
- 분석 결과 조회 로직(api) 개발
    - 시험지 id로 조회시 분석 데이터 json 반환
    - 단원별, 난이도별 문항 수 등 집계
- 분석 결과 시각화 ui 개발
    - json 데이터를 받아 렌더링
 
## 테스트
